## 📚 第一部分：基础知识（必读，2 小时）

### 1.1 什么是微信小程序？

**微信小程序**是一种不需要下载安装即可使用的应用，用户扫一扫或搜一下即可打开。

| 特性 | 说明 | 为什么选择它 |
|------|------|--------------|
| **无需安装** | 用完即走，不占手机空间 | 用户体验好 |
| **开发简单** | 基于 Web 技术（HTML/CSS/JS） | 学习成本低 |
| **生态完善** | 微信提供完整开发工具 | 开发效率高 |
| **用户广泛** | 10 亿 + 微信用户 | 易于推广 |

### 1.2 小程序项目结构

```
miniprogram/
├── app.js              # 小程序逻辑（类似 main.js）
├── app.json            # 小程序配置（页面路由、TabBar 等）
├── app.wxss            # 全局样式（类似 CSS）
├── pages/              # 页面目录
│   ├── index/         # 对话页面
│   ├── appointment/   # 复诊页面
│   ├── knowledge/     # 知识页面
│   └── profile/       # 个人中心
├── utils/              # 工具模块
│   ├── api.js         # 网络请求封装
│   └── util.js        # 通用工具函数
└── images/             # 图片资源
```

### 1.3 小程序文件类型

| 文件后缀 | 说明 | 类似技术 |
|----------|------|----------|
| **.wxml** | 页面结构（类似 HTML） | HTML |
| **.wxss** | 页面样式（类似 CSS） | CSS |
| **.js** | 页面逻辑（类似 JavaScript） | JavaScript |
| **.json** | 页面配置（类似 JSON） | JSON |

### 1.4 核心概念

| 概念 | 说明 | 示例 |
|------|------|------|
| **Page** | 一个页面 | `pages/index/index` |
| **App** | 小程序实例 | `app.js` |
| **setData** | 更新页面数据 | `this.setData({ message: 'hello' })` |
| **onLoad** | 页面加载时执行 | `onLoad(options) {}` |
| **onShow** | 页面显示时执行 | `onShow() {}` |

## 🔌 第五部分：后端接口对接（第 7 天）

### 5.1 网络请求封装

**文件**：`utils/api.js`

**功能**：
- 封装 wx.request
- 自动携带 Token
- 401 自动处理

**关键代码**：

```javascript
// utils/api.js
function request(options) {
  return new Promise((resolve, reject) => {
    const token = wx.getStorageSync('token')
    
    wx.request({
      url: options.url,
      method: options.method || 'GET',
      header: {
        'Content-Type': 'application/json',
        'Authorization': token ? `Bearer ${token}` : ''
      },
      success: (res) => {
        if (res.statusCode === 200) {
          resolve(res.data)
        } else if (res.statusCode === 401) {
          // Token 失效，清除并跳转登录
          wx.removeStorageSync('token')
          wx.reLaunch({ url: '/pages/profile/profile' })
        } else {
          reject(res.data)
        }
      }
    })
  })
}
```

## 🐛 第七部分：常见问题（第 9 天）

### Q1: 小程序启动报错"找不到图标"

**A**: 图标文件缺失，解决：
1. 运行 `python images/generate_icons.py`
2. 或手动添加图标文件到 `images/` 目录

### Q2: 登录失败"网络请求失败"

**A**: 检查：
1. 后端服务是否启动
2. `baseUrl` 配置是否正确
3. 是否勾选"不校验合法域名"

### Q3: 搜索功能返回 401

**A**: API 路径不匹配，修改：
```javascript
// 从 /knowledge/search 改为 /knowledge/search/query
search: (query) => get(`${baseUrl}/knowledge/search/query`, { query })
```

### Q4: 切换用户后聊天记录不清空

**A**: 在 `onShow()` 中检查 sessionId：
```javascript
onShow() {
  const currentSessionId = wx.getStorageSync('sessionId')
  if (currentSessionId !== this.data.currentSessionId) {
    this.setData({ messages: [] })
  }
}
```

### Q5: 真机测试网络错误

**A**: 
1. 使用电脑 IP 替代 localhost
2. 确保手机和电脑同一 WiFi
3. 关闭防火墙或添加 8000 端口规则
