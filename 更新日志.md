## 2026-03-07 - v3.0 最终版本发布

### 新增功能

#### 1. Streamlit 医护后台（✅ 完成）
- **6 个核心页面**：
  - 📊 仪表盘（数据可视化）
  - 👥 患者管理（增删改查 + 筛选）
  - 📅 复诊管理（增删改查 + 状态更新）
  - 💬 对话监管（查看 + 人工接管 + 清理）
  - 📚 知识库管理（增删改查）

- **刷新功能**：所有页面支持刷新获取最新数据

#### 2. 用户管理功能
- **用户注册**：需要管理员密码确认
- **忘记密码**：管理员验证后重置密码
- **密码安全**：bcrypt 加密存储

#### 3. 对话监管增强
- **按会话分组**：相同会话的对话显示在一个折叠框
- **时间排序**：框内正序（从旧到新），框外倒序（最新会话在上）
- **人工接管**：按会话接管，不是单条对话
- **删除会话**：支持删除整个会话的所有对话

#### 4. 实用脚本
- `stop_all.bat` - 停止所有 Python 进程
- `reset_test_data.bat` - 重置测试数据（ID 从 1 开始）

### Bug 修复

1. **端口占用问题**
   - 原因：多个后端实例运行
   - 解决：`stop_all.bat` 强制停止所有 Python 进程

2. **对话数据加载问题**
   - 原因：前端传 `page/page_size`，后端期望 `skip/limit`
   - 解决：API 客户端转换参数

3. **删除功能无效**
   - 原因：确认表单在折叠框内，刷新后不可见
   - 解决：移到折叠框外底部

4. **用户信息显示"未知"**
   - 原因：登录接口未返回用户信息
   - 解决：修改 `/api/auth/login` 返回 `user` 字段
