# Streamlit 后台快速入门指南

> 5 分钟快速启动医护管理后台

---

## 📦 一、安装依赖（首次运行）

### 1. 打开命令行

```bash
# 进入项目目录
cd D:\Project\毕业设计\streamlit_app
```

### 2. 创建虚拟环境

```bash
python -m venv venv
```

### 3. 激活虚拟环境

**Windows：**
```bash
venv\Scripts\activate
```

**Mac/Linux：**
```bash
source venv/bin/activate
```

### 4. 安装依赖

```bash
pip install -r requirements.txt
```

> 💡 **提示**：如果下载慢，可以使用国内镜像：
> ```bash
> pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
> ```

---

## 🚀 二、启动应用

### 方式一：使用启动脚本（推荐）

双击运行：
```
streamlit_app\start.bat
```

### 方式二：手动启动

```bash
# 确保虚拟环境已激活
venv\Scripts\activate

# 启动 Streamlit
streamlit run app.py
```

浏览器会自动打开：http://localhost:8501

## 📊 四、使用指南

### 4.1 仪表盘

查看系统概览数据：
- 患者总数、今日复诊、今日对话
- 近 7 日趋势图表
- 患者性别分布、复诊状态分布

### 4.2 患者管理

1. 点击 **"👥 新增患者"** 创建患者档案
2. 填写姓名、性别、年龄、手机号等信息
3. 点击表格下方的 **"✏️ 编辑"** 修改信息
4. 点击 **"🗑️ 删除"** 删除患者

### 4.3 复诊管理

1. 点击 **"➕ 创建复诊计划"**
2. 选择患者、复诊日期、治疗类型
3. 在复诊列表中点击状态按钮更新状态
4. 支持删除复诊计划

### 4.4 对话监管

1. 查看 AI 与患者的对话记录
2. 按患者筛选对话
3. 点击 **"👤 人工接管"** 进行人工干预

### 4.5 知识库管理

1. 查看知识库条目
2. 按分类筛选
3. 新增/编辑/删除知识条目

## 🐛 五、常见问题

### Q1: 登录失败 "API 请求失败"

**原因**：后端服务未启动

**解决方案**：
```bash
# 启动后端服务
cd D:\Project\毕业设计\backend
python start_server.py
```

### Q2: 端口已被占用

**错误信息**：Port 8501 is already in use

**解决方案**：
```bash
# 使用其他端口
streamlit run app.py --server.port 8502
```

### Q3: 找不到 Python 命令

**原因**：未安装 Python 或未添加到 PATH

**解决方案**：
1. 下载安装 Python 3.10+：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"

### Q4: pip 安装依赖失败

**解决方案**：
```bash
# 升级 pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
