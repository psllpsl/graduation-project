# Streamlit 医护管理后台

> 牙科修复复诊管理系统的医护人员操作平台
> **版本**：v3.1
> **最后更新**：2026 年 3 月 12 日

## 📋 功能特性

- ✅ **仪表盘** - 实时数据可视化看板
- ✅ **患者管理** - 患者档案的增删改查 + 筛选
- ✅ **复诊管理** - 复诊计划的创建与状态管理
- ✅ **对话监管** - 查看 AI 对话记录，支持人工接管
- ✅ **知识库管理** - 专业知识条目的增删改查
- ✅ **系统设置** - 系统参数配置
- ✅ **用户管理** - 注册新用户 + 重置密码

## 🚀 快速开始

### 1. 安装依赖

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境（Windows）
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动应用

**方式一：使用启动脚本（推荐）**

```bash
start.bat
```

**方式二：手动启动**

```bash
venv\Scripts\activate
streamlit run app.py
```

### 3. 停止服务

```bash
# 在项目根目录运行
stop_all.bat
```

### 4. 访问后台

浏览器打开：http://localhost:8501

## 🔧 配置说明

### 后端 API 地址

编辑 `utils/api_client.py` 修改 `BASE_URL`：

```python
BASE_URL = "http://localhost:8000/api"
```

### 主题配置

编辑 `.streamlit/config.toml` 自定义主题：

```toml
[theme]
primaryColor = "#1890FF"
backgroundColor = "#F0F2F5"
```

## 🐛 常见问题

### Q1: 启动时报错 "Port already in use"

**解决方案**：
```bash
# 运行停止脚本
stop_all.bat

# 或使用其他端口
streamlit run app.py --server.port 8502
```

### Q2: 页面空白或无数据

**检查项**：
1. 后端服务是否启动（http://localhost:8000/docs）
2. API 地址是否正确
3. 是否已登录

### Q3: 中文显示乱码

**解决方案**：
确保文件保存为 UTF-8 编码。
