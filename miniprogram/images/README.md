# 微信小程序图标说明

## 需要的图标文件

请将以下图标文件放置在此目录下：

### TabBar 图标（建议尺寸：81x81px）

| 文件名 | 说明 | 颜色 |
|--------|------|------|
| `chat.png` | 对话图标（未选中） | #999999 |
| `chat-active.png` | 对话图标（选中） | #4A90D9 |
| `appointment.png` | 复诊图标（未选中） | #999999 |
| `appointment-active.png` | 复诊图标（选中） | #4A90D9 |
| `knowledge.png` | 知识图标（未选中） | #999999 |
| `knowledge-active.png` | 知识图标（选中） | #4A90D9 |
| `profile.png` | 我的图标（未选中） | #999999 |
| `profile-active.png` | 我的图标（选中） | #4A90D9 |

### 其他图标

| 文件名 | 说明 | 尺寸 |
|--------|------|------|
| `ai-avatar.png` | AI 助手头像 | 200x200px |
| `user-avatar.png` | 用户默认头像 | 200x200px |

## 图标制作工具推荐

1. **在线工具**：
   - IconFont: https://www.iconfont.cn/
   - IconPark: https://iconpark.oceanengine.com/
   - 稿定设计：https://www.gaoding.com/

2. **设计软件**：
   - Figma（免费）
   - Sketch
   - Adobe Illustrator

3. **小程序图标生成器**：
   - 微信开发者工具自带图标裁剪功能

## 临时方案

如果暂时没有图标，可以：
1. 使用 emoji 表情作为临时图标
2. 使用纯色背景 + 文字的图片
3. 从开源图标库下载类似图标

## 注意事项

1. 图标必须是 PNG 格式
2. TabBar 图标大小建议为 81x81 像素
3. 图标文件不能大于 40KB
4. 图标应该是正方形
