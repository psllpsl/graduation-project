## 🚀 快速开始

### 1. 安装微信开发者工具

下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download

选择稳定版（Stable Build）下载并安装。

### 2. 导入项目

1. 打开微信开发者工具
2. 点击 "+" 导入项目
3. 项目目录：`miniprogram/`
4. 填写 AppID（测试可使用测试号）

### 3. 配置网络地址

编辑 `app.js`，修改 `baseUrl` 为你的电脑 IP：

```javascript
const baseUrl = 'http://192.168.x.x:8000/api'  // 替换为你的电脑 IP
```

> 💡 **查看电脑 IP**：
> ```bash
> ipconfig  # Windows
> ```

### 4. 配置不校验域名

1. 微信开发者工具 → 详情
2. 本地设置
3. ✅ 勾选 "不校验合法域名"

### 5. 编译运行

1. 点击 "编译" 按钮
2. 查看模拟器效果
3. 点击 "真机调试" 在手机上测试

## 🐛 常见问题

### Q1: 网络请求失败

**原因**：IP 地址错误或防火墙阻止

**解决方案**：
1. 修改 `app.js` 中的 `baseUrl` 为正确的电脑 IP
2. 确保手机和电脑在同一 WiFi
3. 关闭防火墙或添加 8000 端口规则

### Q2: 登录失败

**检查项**：
1. 后端服务是否启动
2. 患者档案是否存在
3. IP 地址是否正确

### Q3: 图标缺失

**解决方案**：
确保 `images/` 目录下的图标文件存在。
