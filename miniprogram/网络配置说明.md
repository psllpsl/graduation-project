# 小程序网络配置说明

## 🚨 登录失败：ERR_CONNECTION_TIMED_OUT

这个错误表示小程序无法连接到后端 API。

---

## 🔧 解决方案

### 步骤 1：检查 IP 地址

1. **查看当前电脑 IP**：
   ```bash
   ipconfig
   ```
   
2. **找到 IPv4 地址**（通常是 `192.168.x.x` 或 `10.x.x.x`）

3. **修改小程序配置**：
   - 打开 `miniprogram/app.js`
   - 找到 `baseUrl` 配置
   - 修改为你的电脑 IP：
   ```javascript
   baseUrl: 'http://192.168.0.103:8000/api'  // 替换为你的 IP
   ```

### 步骤 2：配置防火墙

**方式一：运行脚本（推荐）**

1. 右键点击 `setup_firewall.bat`
2. 选择 **"以管理员身份运行"**
3. 等待完成提示

**方式二：手动添加规则**

1. 打开 **Windows Defender 防火墙**
2. 高级设置 → 入站规则 → 新建规则
3. 端口 → TCP → 特定本地端口：`8000`
4. 允许连接 → 完成

### 步骤 3：微信开发者工具配置

1. 点击右上角 **"详情"**
2. **本地设置** 标签
3. ✅ 勾选 **"不校验合法域名"**
4. ✅ 勾选 **"不校验 TLS 版本"**

### 步骤 4：重启服务

```bash
# 重启后端
cd backend
venv\Scripts\activate
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 重启小程序
# 微信开发者工具 → 编译
```

## 🔄 网络变化时的处理

如果更换了 WiFi 或网络环境，IP 地址可能会变化：

1. 重新运行 `ipconfig` 查看新 IP
2. 更新 `miniprogram/app.js` 中的 `baseUrl`
3. 重启小程序

## ⚠️ 常见问题

### Q1: 小程序还是连接失败

**检查项**：
- [ ] 手机和电脑在同一 WiFi
- [ ] 防火墙规则已添加
- [ ] 后端服务正在运行
- [ ] IP 地址配置正确

### Q2: 模拟器可以，真机不行

**原因**：真机需要使用局域网 IP，不能用 localhost

**解决**：
- 模拟器：`baseUrl: 'http://localhost:8000/api'`
- 真机：`baseUrl: 'http://你的电脑 IP:8000/api'`

### Q3: 每次都要修改 IP 很麻烦

**解决**：
- 使用路由器分配的固定 IP
- 或者使用内网穿透工具（如 ngrok）
