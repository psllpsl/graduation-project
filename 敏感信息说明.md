# 敏感信息说明

本文档包含的敏感信息已在上传 GitHub 前进行脱敏处理。

## 需要自行配置的信息

### 1. 微信小程序 AppID

在以下文件中替换为你的测试 AppID：
- `miniprogram/project.config.json`
- `miniprogram/README.md`

```json
{
  "appid": "你的 AppID"
}
```

### 2. AutoDL AI 服务地址

AutoDL 实例地址每次重启会变化，请根据实际情况配置：

**格式**：`https://实例 ID.seetacloud.com:端口/generate`

**配置方法**：
1. 运行 `backend/start_server.py`
2. 输入你的 AutoDL 公网地址
3. 自动更新 `.env` 文件

### 3. 数据库密码

在以下文件中替换为你的 MySQL 密码：
- `backend/.env`
- `backend/app/config.py`（仅示例）

```env
DATABASE_PASSWORD=你的密码
```

### 4. JWT 密钥

生产环境请修改为随机字符串：

```env
SECRET_KEY=你的随机密钥（至少 32 字符）
```

**生成随机密钥**：
```python
import secrets
print(secrets.token_urlsafe(32))
```

## 敏感信息处理

已上传到 GitHub 的文档中，以下信息已脱敏：

| 信息类型 | 脱敏方式 |
|----------|----------|
| AppID | `your-appid-here` → `你的 AppID` |
| AutoDL 地址 | 完整 URL → `https://实例 ID.seetacloud.com:端口` |
| 数据库密码 | 实际密码 → `你的密码` |
| JWT 密钥 | 实际密钥 → `your-secret-key` |

---

## 安全建议

1. **不要上传 `.env` 文件** - 已添加到 `.gitignore`
2. **不要上传真实密码** - 文档中使用占位符
3. **生产环境使用 HTTPS** - 保护数据传输
4. **定期更换密钥** - 提高安全性
