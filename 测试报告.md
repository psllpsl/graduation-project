## 测试概述

### 测试目标

验证"牙科修复复诊提醒与管理系统"的功能完整性、性能指标和安全性，确保系统满足毕业设计要求和实际使用需求。

### 测试范围

| 模块 | 测试内容 |
|------|----------|
| **后端 API** | 认证、患者管理、复诊管理、对话管理、知识库、数据统计 |
| **AI 服务** | 对话生成、知识检索、RAG 增强 |
| **小程序前端** | 登录、对话、复诊、知识库、个人中心 |
| **数据库** | 数据完整性、查询性能、事务处理 |

### 测试方法

- **单元测试**：针对独立功能模块进行测试
- **接口测试**：使用 Postman/Swagger 测试 API 接口
- **集成测试**：测试模块间的数据流转
- **UI 测试**：人工测试小程序界面交互
- **性能测试**：压力测试和响应时间测试
