## 📊 项目概览

### 系统定位

这是一个**医疗健康类软件系统**，为牙科修复患者提供 7×24 小时智能随访服务，通过 AI 客服降低医护工作负担，提升患者治疗依从性。

### 核心价值

1. **降低漏复诊率** - 自动化复诊提醒 + 专业术后指导
2. **减轻医护负担** - AI 客服替代重复性电话咨询
3. **提升患者体验** - 微信小程序 + 即时专业回复

## 📞 快速启动

### 停止所有服务
```bash
stop_all.bat
```

### 启动后端
```bash
cd backend
start_server.bat
```

### 启动 Streamlit
```bash
cd streamlit_app
start.bat
```

### 访问地址
- **后端 API**：http://localhost:8000/docs
- **Streamlit**：http://localhost:8501
- **小程序**：微信开发者工具
