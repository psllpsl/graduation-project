# AutoDL 快速启动指南（Transformers 版本）

## 📁 上传文件

在 AutoDL JupyterLab 中，上传以下文件到 `/root/autodl-tmp/`：

- `start.sh` - 一键启动脚本 ⭐
- `start_inference.py` - 推理服务脚本
- `.autodl_config` - 配置文件（如果之前有）

---

## 🚀 启动步骤

### 步骤 1：赋予权限

```bash
chmod +x start.sh
```

### 步骤 2：运行脚本

```bash
bash start.sh
```

---

## 📋 脚本会自动完成

✅ 自动检测模型路径（`./models/dental_qwen_merged`）  
✅ 自动读取/保存配置（公网地址、端口）  
✅ 自动安装依赖（transformers、fastapi、torch）  
✅ 自动启动推理服务  
✅ 自动测试功能  

---

## 🔄 换机器后使用

```bash
bash start.sh
```

当询问"是否更换了 AutoDL 实例"时：

- **直接回车** → 使用原配置启动
- **输入 y** → 重新输入公网地址

---

## 📝 配置说明

脚本会创建 `.autodl_config` 文件，保存：
- 模型路径
- 服务端口
- 公网地址
- 进程 PID

**下次运行自动读取，无需重复输入！**

---

## 🧪 测试命令

```bash
# 健康检查
curl http://localhost:6008/health

# 推理测试
curl http://localhost:6008/generate \
    -H "Content-Type: application/json" \
    -d '{"prompt": "种植牙术后多久能吃饭？", "max_tokens": 150}'

# 查看日志
tail -f inference.log
```

---

## 🌐 公网访问配置

1. 在 AutoDL 控制台 → 自定义服务 → 添加服务
2. 填写端口 `6008`，协议 `HTTP`
3. 复制生成的公网 URL（如：`https://你的实例 ID...seetacloud.com:8443`）
4. 重新运行 `bash start.sh`，输入公网地址

---

## 📊 常用命令

```bash
# 查看配置
cat .autodl_config

# 查看进程
ps aux | grep start_inference

# 停止服务
kill $(grep PID .autodl_config | cut -d'=' -f2)

# 重启服务
bash start.sh

# 查看日志
tail -f inference.log
```
