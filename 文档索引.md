## 📞 常用命令

### 启动服务
```bash
# 停止所有服务
stop_all.bat

# 启动后端
cd backend
start_server.bat

# 启动 Streamlit
cd streamlit_app
start.bat
```

- **后端 API**：http://localhost:8000/docs
- **Streamlit**：http://localhost:8501
